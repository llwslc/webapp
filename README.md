## Installation

 * node
 * npm
 * vue-cli
  * npm i vue-cli -g


## Build Setup

``` bash
# init project
vue init llwslc/webapp my-project

# install dependencies
cd my-project
npm i

# run your client
npm run dev

# run your server
npm run start
```
